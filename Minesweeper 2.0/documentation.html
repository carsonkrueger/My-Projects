The board is styled by a css file called styles.css and uses javascript to style as well. Javascript also creates elements and gives elements images.

User will use the RESET button to reset the board, as a form of input.

User will click on tiles to play the game, as a form of input.

As User clicks on the tiles, the board will dynamically modify based upon clicks.

A httprequest will be used upon loading the game. Displaying the contents inside of "gameBoardy.json".

Local storage is used to store the time the user starts the game.

The user will win once the tiles without bombs are uncovered and all the tiles with bombs underneath them, are flagged. Winning or losing will prevent the user from modifying the gameboard and will force them to reset the board.
